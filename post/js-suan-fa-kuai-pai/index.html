js算法：快速排序

2021-03-23 算法 js

个人认为，快速排序的基本思想就是： 将需要排序的数切分成两段，一段小，一段大，通过递归不断切分，从而完成了排序。

具体步骤如下：

  1. 从需要排序的数组中随便选择一个数，作为中间数 ，保存到一个变量 centerValue 中。不过一般为了方便，可直接选择第一个数或者中间那个数为中间数。
  2. 从需要排序的数组里删除这个中间数。
  3. 定义两个新数组，左数组 leftArray 和右数组 rightArray 。
  4. 遍历数组，依次与中间数 centerValue 比较大小。 • 比 centerValue 小的放左数组 leftArray ； • 比 centerValue 大的放右数组 rightArray ；
  5. 使用递归的方式，按以上规则继续对左数组leftArray 和右数组 rightArray 进行划分。
  6. 返回排序结果。
  7. 为防止递归陷入死循环，需要在函数开头添加一个判断条件，即：如果数组的长度小于等于1，则将这个数组return回去。
function quickSort(array) {
    // 为防止递归陷入死循环
    if(array.length <= 1) {
        return array
    }
    let leftArr = [],
        rightArr = [],
        centerVal = array.shift()
    
    array.forEach(item => {
        if(item < centerVal) {
            // 如果小于中间值 则push到leftArr里面
            leftArr.push(item)
        }else {
            // 否则push到rightArr里面
            rightArr.push(item)
        }
    });
    
    let leftArrSortRes = quickSort(leftArr)
    let rightArrSortRes = quickSort(rightArr)
    return leftArrSortRes.concat(centerVal, rightArrSortRes)
    // 等同于上面的代码
    // return quickSort(leftArr).concat(centerVal, quickSort(rightArr))
}

let sortArr = quickSort([52,45,4,2,64,21,652,15,1,0,-54,-1,58,7,9])
console.log(sortArr); // [-54, -1, 0, 1, 2, 4, 7, 9, 15, 21, 45, 52, 58, 64, 652]
