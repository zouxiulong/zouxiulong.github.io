算法：冒泡排序

2021-02-25 算法

对数组进行 冒泡排序 算是比较简单的，冒泡排序也是容易理解的一种排序算法了，在面试的时候，很可能就会问到。

实现原理

数组有 n 个数，比较每相邻的两个数，如果前者比后者大，则把两个数位置进行位置交换。一轮下来就可以选出最大的一个数放在最后面；经过 n 轮比较就完成了所有数的排序。 如图所示：

话不多说，直接上代码

let arr = [0,5,9,6,4,27,21,10,9,-6,5,15,-10]
for(let i = 0; i < arr.length - 1; i++ ) {
    // 如果前一个数 大于 后一个数则交换位置
    if(arr[i] > arr[i+1]) {
        let max = arr[i] // 声明一个变量暂存数据
        arr[i] = arr[i+1]
        arr[i+1] = max
    }
}
console.log(arr)  // [0, 5, 6, 4, 9, 21, 10, 9, -6, 5, 15, -10, 27]  // 第一轮得出最大值

这里有一个难点，就是 i < arr.length - 1 ，是因为倒数第二轮进行比较的时候已经得出了最大值，所以 arr.length 可以进减1。

我们继续完善代码

let arr = [0,5,9,6,4,27,21,10,9,-6,5,15,-10]
// 遍历数组，次数就是arr.length - 1
for (let j = 0; j < arr.length - 1; j++) {
    
    for(let i = 0; i < arr.length - 1; i++ ) {
        console.log(arr[i] , arr[i+1]);
        if(arr[i] > arr[i+1]) {
            let max = arr[i]
            arr[i] = arr[i+1]
            arr[i+1] = max
        }
    }
    
}
console.log(arr)  // [-10, -6, 0, 4, 5, 5, 6, 9, 9, 10, 15, 21, 27]

我们能找到数组中最大的数，放到最后，这样重复 arr.length - 1 次，便可以实现数组按从小到大的顺序排好了。为什么是 arr.length - 1 次 ，因为倒数第二轮已经拍好了顺序，这里我们可以进行减1，减少不必要的循环。

虽然上面的代码已经实现冒泡排序了，但就像注释中提到的，内层 for 循环的次数写成，i < arr.length - 1 ，是不是合适呢？ 我们想一下，当第一次，找到最大数，放到最后，那么下一次，遍历的时候，是不是就不能把最后一个数算上了呢？因为他就是最大的了，不会出现，前一个数比后一个数大，要交换位置的情况，所以内层 for 循环的次数，改成 i < arr.length - 1 - j ，才合适，看下面的代码。

let arr = [0,5,9,6,4,27,21,10,9,-6,5,15,-10]
for (let j = 0; j < arr.length - 1; j++) {
    // 这里要根据外层for循环的 j，逐渐减少内层 for循环的次数
    for(let i = 0; i < arr.length - 1 - j; i++ ) {
        if(arr[i] > arr[i+1]) {
            let max = arr[i]
            arr[i] = arr[i+1]
            arr[i+1] = max
        }
    }
}

我们想下有种情况，当原数组是：arr = [1,2,4,3] 在经过第一轮冒泡排序之后，数组就变成了：arr = [1,2,3,4] 此时，数组已经排序完成了，但是按上面的代码来看，数组还会继续排序，所以我们加一个标志位，如果某次循环完后，没有任何两数进行交换，就将标志位 设置为 true，表示排序完成，这样我们就可以减少不必要的排序，提高性能。

终极代码：

let arr = [1,2,4,3]
for (let j = 0; j < arr.length - 1; j++) {
    // 声明一个变量，作为标志位
    let flag = true
    for(let i = 0; i < arr.length - 1 - j; i++ ) {
        if(arr[i] > arr[i+1]) {
            let max = arr[i] 
            arr[i] = arr[i+1]
            arr[i+1] = max
						flag = false
        }
    }
    if(flag) break
}

总结

1、外层 for 循环控制循环次数 2、内层 for 循环进行两数交换，找每次的最大数，排到最后 3、设置一个标志位，减少不必要的循环
