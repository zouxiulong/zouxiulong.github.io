算法：二分查找

2021-03-24

二分查找也称为折半查找，是指在有序的数组里找出指定的值，返回该值在数组中的索引。

二分查找

二分查找也称折半查找（Binary Search），是一种在有序数组中查找某一特定元素的搜索算法。我们可以从定义可知，运用二分搜索的前提是数组必须是有序的，这里需要注意的是，我们的输入不一定是数组，也可以是数组中某一区间的起始位置和终止位置。

注：二分查找必须在有序数组中才有效，无序的数组不能实现查找功能。比如：在[10,5,6,7,8,9,20]中查找10，中间索引位置的值为7，比较得出7比10小，因而应该在右子数组中查找，实际上不可能找到10；

我们通过下面的栗子来了解二分查找具体是怎么实现的，首先我们先定义一个数组，查询元素 8 的索引。

let array = [1,3,4,5,6,8,12,14,16]  // 查找元素为8的索引

1. 我们先定义两个指针分别指向数组的头部和尾部，这是我们在整个数组中查找的范围，当我们在数组某一区间进行查询时，可以输入数组，起始位置，终止位置进行查询。如下图



2. 我们需要找出mid，也就是 mid =（left + right）/ 2，通过首末索引相加来找到中间索引，通过中间索引判断目标元素所在的区域，注意（left + right）/ 2 的结果可能会出现有小数点的情况，我们需要对它进行向下取整，也就是Math.floor( (left + right) / 2 )



3. 此时我们的 mid = 4，array[mid] = 6 < target,那么我们需要移动我们的 left 指针，让left = mid + 1，下次则可以在新的 left 和 right 区间内搜索目标值，下图为移动前和移动后



4. 我们需要在 left 和 right 之间重新计算 mid 值，此时的 mid = Math.floor( (8 + 5) / 2 ) = 6 然后将 array[mid] 与 target 继续比较，进而决定下次移动left 指针还是 right 指针，见下图



5.我们发现 array[mid] > target，则需要移动我们的 right 指针， 则 right = mid - 1；则移动过后我们的 left 和 right 会重合，这里是我们的一个重点大家需要注意一下，后面会对此做详细叙述。



6.我们需要在 left 和 right 之间继续计算 mid 值，则 mid = Math.floor( (5 +5) / 2 ) = 5 ，见下图，此时我们将 array[mid] 和 target 比较，则发现两值相等，返回 mid 即可，如果不相等则跳出循环，返回 -1。



二分查找的执行过程如下

  1. 首先要拿到有序数组的首末索引
  2. 通过首末索引相加来找到中间索引，通过中间索引判断目标元素所在的区域
  3. 如果碰巧mid对应的数就是目标，那就直接返回该索引
  4. 如果mid索引对应的元素大于target，那么证明target，在以mid分割的数组的前半部分，所以要让末尾索引变成 mid - 1，相反就让首索引变成 mid + 1
  5. 查找结束条件，1.就是成功找到目标元素，然后返回相对应的索引，2.没有查找到目标元素，返回-1

下面我们来看一下二分查找的代码

//array:数组  target:查找的元素
function search(array, target) {
    // 初始索引开始位置和结束位置
    let left = 0,
        right = array.length - 1
    while(left <= right) {
        // 取中间的索引
        let mid = Math.floor((left + right) / 2)
        if(array[mid] === target) {
            // 如果找到则直接返回
            return mid
        }else if(array[mid] > target) {
            // 如果target是小于数组中间索引的值则将索引结束位置设置为中间索引-1
            right = mid - 1
        }else {
            // 如果target是大雨数组中间索引的值则将索引结束位置设置为中间索引+1
            left = mid + 1
        }
    }
    // 如果在循环内没有找到查找的target (left <= right) 的情况则返回-1
    return -1
}
let array = [1,3,4,5,6,8,12,14,16]
console.log(search(array, 2)); // -1
console.log(search(array, 3)); // 1

递归版本:

// array:数组 target:查找的元素 left:开始索引 right:结束索引
function search(array, target, left, right) {
    // 首先判断当前起始索引是否大于结束索引,如果大于说明没有找到元素返回-1
    if(left > right)  {
        return -1
    }
    // 如果手动调用不写left和right参数会当做第一次运行默认值
    // 三元表达式:如果不写right参数则为undefined说明第一次调用所以结束索引为array.length-1
    // 如果是递归调用则使用传进来的参数right值, 
    // left同理right
    var left = left == undefined ? 0 : left
    var right = right == undefined ? array.length - 1 : right 
    var mid = Math.floor((left + right) / 2)
    
    if(array[mid] === target) {
        return mid
    }else if(array[mid] > target) {
        right = mid - 1
        return search(array, target, left, right)
    }else {
        left = mid + 1
        return search(array, target, left, right)
    }
}
console.log(search(array, 2)); // -1
console.log(search(array, 3)); // 1
